# 🧠 MindLens – AI-Powered Wellbeing Watch

MindLens is an AI-powered mental wellbeing platform designed to help users monitor their emotional health, identify burnout risks, and gain meaningful insights through intelligent analysis and interactive tools.

The project aims to make mental wellbeing tracking accessible, proactive, and user-friendly.

---

## 🌐 Live Demo

🔗 https://mindlens-cyan.vercel.app

---

## 📌 Features

### 📝 Daily Mood & Stress Tracking

Users can regularly log their mood and stress levels to build healthy self-awareness habits.

### 🤖 AI-Powered Sentiment Analysis

Analyzes journal entries and user inputs to determine emotional trends and generate insights.

### 📊 Burnout Risk Predictor

Evaluates behavioural patterns and provides an estimation of burnout risk.

### 😊 Emotion Analysis

Detects emotional patterns and helps users understand their mental state.

### 🧠 Mental Health Questionnaire

Interactive questionnaires help assess overall wellbeing.

### 💬 AI Support Chat

Provides supportive responses and guidance through an AI-powered assistant.

### 📈 Insights Dashboard

Visualizes user progress and trends over time.

## 🚀 Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Mindlens---Dhruv-and-Harshit.git
```

### 2. Navigate to the project directory

```bash
cd Mindlens---Dhruv-and-Harshit
```

### 3. Install dependencies

```bash
npm install
```

### 4. Configure environment variables

Create a `.env` file and add:

```env
VITE_SUPABASE_PROJECT_ID=your_project_id
VITE_SUPABASE_PUBLISHABLE_KEY=your_publishable_key
VITE_SUPABASE_URL=your_supabase_url
```

### 5. Run the development server

```bash
npm run dev
```
